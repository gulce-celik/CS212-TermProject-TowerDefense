# Tower Defense Game (CS212 Final Project)

A 2D Tower Defense game developed in Java using Swing for the **CSE 212 - Software Development** course. Defend your base against waves of enemies by strategically placing towers!

    -   **Big Rocket** (1200$): Massive damage for tough enemies.
-   **Challenging Enemies**:
    -   Green, White, and Orange enemy variants with different speeds and health.
    -   **Boss Tank**: A heavily armored enemy that requires serious firepower.
-   **Game Mechanics**:
    -   **Economy System**: Earn money by defeating enemies to buy upgrades.
    -   **Wave System**: Progressively harder waves of enemies.
    -   **Health & Score**: Track your performance and aim for the high score.
    -   **Sell Mechanic**: Sell misplaced towers to recover funds.
    -   **Pause/Resume**: Control the flow of the game.
